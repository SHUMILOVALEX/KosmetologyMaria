# MariaKosmetics — сайт мастера-косметолога

Статический сайт (HTML/CSS/JS, без бэкенда). Работает полностью локально — открывается прямо в браузере.

## Как открыть

Дважды кликните `index.html` — сайт откроется в браузере по умолчанию. Либо перетащите файл в окно браузера. Интернет нужен только для загрузки шрифта Manrope с Google Fonts — без сети сайт тоже откроется, просто шрифт подменится на системный.

## Структура

```
index.html                     Главная
services.html                  Мои услуги

reviews.html                   Отзывы клиентов
blog.html                      Блог (список статей)
blog-post-home-care.html       Статья: как выбрать домашний уход
blog-post-peeling-myths.html   Статья: мифы о пилингах
blog-post-acne.html            Статья: обострение акне

js/main.js                     Логика: данные студии, меню, анимации, фильтры, форма записи
```

## Как поменять имя, город, телефон, соцсети

Откройте `js/main.js`, в самом начале файла — объект `CONFIG`:

```js
const CONFIG = {
  studioName:   'MariaKosmetics',
  masterName:   'Мария Шумилова',
  city:         'Челябинск',
  instagram:    '@mari_shymi',
  telegram:     '@mari_shymi',
  vk:           'mari2406ptrv',
  ...
};
```

Поменяйте значения — они автоматически подставятся на **всех** страницах сайта (в шапке, футере, блоке «О мастере» и т.д.). Отдельно редактировать HTML для этого не нужно.

## Как заменить фото-плейсхолдеры на настоящие фото

Сейчас вместо фото используются стилизованные градиентные блоки с подписью, например:

```html
<div class="ph portrait"><p class="ph-label">Фото: портрет мастера</p></div>
```

Чтобы поставить реальное фото, замените такой блок на обычный `<img>`, например:

```html
<img src="images/portrait.jpg" alt="Мария Шумилова, косметолог" style="border-radius:16px; aspect-ratio:3/4; object-fit:cover;">
```

Создайте папку `images/` рядом с HTML-файлами и положите туда свои фотографии.

## Важно про форму записи (booking.html)

Форма записи работает только на фронтенде: после нажатия «Подтвердить запись» она просто показывает экран «Спасибо» — **никакие данные никуда не отправляются и не сохраняются**. Это сделано специально, чтобы не создавать у клиентов ложное ощущение, будто заявка дошла.

Чтобы форма реально работала, нужно выбрать один из вариантов:
- Подключить сервис форм (например, Formspree, Getform) — потребует минимальных правок в `js/main.js`.
- Настроить приём заявок в Telegram-бота.
- Интегрировать с CRM/YClients/Altegio, если вы там ведёте запись.
- Поднять простой бэкенд (например, на Node.js), который получает данные формы и пишет в почту/базу.

## Если понадобится публичная ссылка на сайт

Сейчас сайт — это просто набор файлов на компьютере. Чтобы получить ссылку, которой можно поделиться, самый простой бесплатный вариант — GitHub Pages или Netlify Drop (достаточно перетащить папку с сайтом на сайт Netlify). Если понадобится помощь с деплоем — просто попросите.

## Технические детали

- Без сборки и зависимостей — чистые HTML/CSS/JS, ничего устанавливать не нужно.
- Шрифт: Manrope (Google Fonts) с fallback на системный шрифт.
- Анимации появления при скролле — `IntersectionObserver`, работает во всех современных браузерах.
- Мобильная адаптация — брейкпоинты на 960px и 640px, проверено на ширине от 360px.
