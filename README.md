# MariaKosmetics — сайт мастера-косметолога

Статический сайт (HTML/CSS/JS, без бэкенда). Работает полностью локально — открывается прямо в браузере.

## Как открыть

Дважды кликните `index.html` — сайт откроется в браузере по умолчанию. Либо перетащите файл в окно браузера. Интернет нужен только для загрузки шрифта Manrope с Google Fonts — без сети сайт тоже откроется, просто шрифт подменится на системный.

## Как поменять имя, город, телефон, соцсети

Откройте `js/main.js`, в самом начале файла — объект `CONFIG`:

```js
const CONFIG = {
  studioName:   'MariaKosmetics',
  masterName:   'Мария Шумилова',
  city:         'Челябинск',
  instagram:    '@mari_shymi',
  telegram:     '@mari_shymi',
  vk:           'mari2406ptrv',
  ...
};
```

Поменяйте значения — они автоматически подставятся на **всех** страницах сайта (в шапке, футере, блоке «О специалисте» и т.д.). Отдельно редактировать HTML для этого не нужно.

## Как заменить фото-плейсхолдеры на настоящие фото

Сейчас вместо фото используются стилизованные градиентные блоки с подписью, например:

```html
<div class="ph portrait"><p class="ph-label">Фото: портрет мастера</p></div>
```

Чтобы поставить реальное фото, замените такой блок на обычный `<img>`, например:

```html
<img src="images/portrait.jpg" alt="Мария Шумилова, косметолог" style="border-radius:16px; aspect-ratio:3/4; object-fit:cover;">
```

Создайте папку `images/` рядом с HTML-файлами и положите туда свои фотографии.

## Форма записи (booking.html) — заявки приходят в Telegram

При отправке формы данные (услуга, дата, время, имя, телефон, дата рождения, отмеченные проблемы кожи) уходят на серверную функцию `netlify/functions/book.js`, которая пересылает их мастеру сообщением в Telegram через Bot API.

Чтобы это заработало, сайт должен быть развёрнут на **Netlify** (не GitHub Pages — там нет серверных функций), и нужно один раз настроить бота:

1. **Создать Telegram-бота.** Написать в Telegram боту [@BotFather](https://t.me/BotFather) команду `/newbot`, следовать инструкциям — в конце он даст **токен** вида `123456:ABC-...`.
2. **Узнать свой chat_id.** Написать что-нибудь новому боту (просто откройте с ним диалог и отправьте любое сообщение), затем открыть в браузере:
   `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
   и найти в ответе `"chat":{"id": ЧИСЛО, ...}` — это и есть `chat_id`.
3. **Задеплоить сайт на Netlify:** зайти на [netlify.com](https://netlify.com), подключить этот GitHub-репозиторий («Add new site» → «Import an existing project»). Функции подхватятся автоматически (см. `netlify.toml`).
4. **Добавить переменные окружения** в настройках сайта на Netlify (Site configuration → Environment variables):
   - `TELEGRAM_BOT_TOKEN` — токен бота из шага 1
   - `TELEGRAM_CHAT_ID` — chat_id из шага 2
5. Передеплоить сайт (Netlify сделает это сам после сохранения переменных, либо нажать «Trigger deploy»).

После этого при отправке формы заявка будет мгновенно приходить сообщением в чат с ботом.

Если вместо Telegram понадобится email или CRM (YClients/Altegio) — можно заменить содержимое `netlify/functions/book.js` под нужный сервис, форма на фронтенде менять не придётся.

## Если понадобится публичная ссылка на сайт

Из-за формы записи (см. выше) сайт нужно разместить именно на **Netlify** — просто подключите GitHub-репозиторий на [netlify.com](https://netlify.com), это бесплатно. GitHub Pages в этом случае не подойдёт, так как не поддерживает серверные функции.

## Технические детали

- Без сборки и зависимостей — чистые HTML/CSS/JS, ничего устанавливать не нужно.
- Шрифт: Manrope (Google Fonts) с fallback на системный шрифт.
- Анимации появления при скролле — `IntersectionObserver`, работает во всех современных браузерах.
- Мобильная адаптация — брейкпоинты на 960px и 640px, проверено на ширине от 360px.
